## 프로젝트 소개
> Rebread란?
* 어원 그대로 빵을 재사용한다는 의미로써, 최대한 남는 빵을 줄이고 할인된 가격에 만나볼수 있다.

<br><br>

> 개발목표
* 매년 증가하는 로스빵(팔리지 않고 남은 빵)을 할인된 가격에서 판매할 수 있게 중개해주는 플랫폼 
* 로스빵과 일반빵으로 나뉘고 판매금액의 일정액은 마일리지로 전환되어 해피빈과 유사하게 기부도 가능한 플랫폼
